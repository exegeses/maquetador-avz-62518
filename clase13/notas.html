Using CSS transitions

  • CSS transitions proveen la manera de controlar la velocidad de uno o varios cambios de estado de una o varias propiedades CSS.
  • En vez de implementar los cambios de estado de manera inmediata, podemos implementarlos durante un lapso (o incluso luego de un lapso). Por ejemplo, si cambiamos el color de un elemento de un color A a un color B, sin una transición, el cambio es inmediato.
    Pero con CSS transitions, los cambios ocurren a intervalos que pueden seguir una curva de aceleración que podemos personalizar.
  • Las CSS transitions nos dejan decidir
    • qué propiedades modificar (listándolas de manera explçicita),
    • cuándo comenzará la transición (seteando el delay),
    • cuánto tiempo llevará la transición (seteando la duración),
    • y cómo sucederá la transición (seteando la función de aceleración).
